# Simple Harmonic Motion Graph Maker

This script generates the graphs according to the laws of Simple Harmonic Motion, from physics. You can set the necessary initial condition in the variables in code. It assumes a mass-spring system.

This was an exercise for personal study reasons in my Physics class.
